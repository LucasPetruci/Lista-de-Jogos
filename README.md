# Lista-de-Jogos
Uma aplicação Web API REST de uma Lista de Jogos feita em Spring, onde esses jogos possuem suas informações e um sistema de classificação de acordo com o seu estilo. 

## Tecnologias Utilizadas
* Spring REST
* JPA
* Hibernate
* H2
* PostgreSQL
* Docker
